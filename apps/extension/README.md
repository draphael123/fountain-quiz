# Fountain Quiz Chrome Extension

Chrome extension for quick quiz and search functionality.

## Installation

1. Open Chrome and navigate to `chrome://extensions/`
2. Enable "Developer mode"
3. Click "Load unpacked"
4. Select the `apps/extension` directory

## Features

- Quick 5-question quiz
- Search knowledge base
- Customizable settings (tags, difficulty, question count)

## Development

The extension communicates with the web app API. Make sure to update the `API_BASE` URL in `popup.js` for production.
